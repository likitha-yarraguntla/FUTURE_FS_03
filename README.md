## 📌 Project Overview

Sri Balaji Ghee Sweets is a professional website developed for a local sweet shop in Sattenapalli. The website helps customers explore sweets, view products, add items to cart, and place orders through WhatsApp.

This project was developed as part of the **Future Interns Full Stack Web Development Internship – Task 03 (Local Business Website & Live Pitch Project).**
